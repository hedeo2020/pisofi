PisoNet Orange Pi One bootstrap agent

Do not install over the existing LPB service yet.

Armbian / Debian systemd install:

1. Copy pisonet-agent to /usr/local/bin/pisonet-agent and chmod 0755.
2. Run: /usr/local/bin/pisonet-agent -self-test
3. Create /etc/pisonet-agent and copy config.json.example to config.json.
4. Replace the server URL, provisioned device ID, and unique device secret; chmod 0600 config.json.
5. Create /var/lib/pisonet-agent/spool.
6. Copy pisonet-agent.service to /etc/systemd/system/pisonet-agent.service.
7. Run: systemctl daemon-reload
8. Run: systemctl enable --now pisonet-agent
9. Inspect: journalctl -u pisonet-agent -f

OpenWrt/procd install:

1. Copy pisonet-agent to /usr/bin/pisonet-agent and chmod 0755.
2. Run: /usr/bin/pisonet-agent -self-test
3. Create /etc/pisonet-agent and copy config.json.example to config.json.
4. Replace the server URL, provisioned device ID, and unique device secret; chmod 0600 config.json.
5. Copy pisonet-agent.init to /etc/init.d/pisonet-agent and chmod 0755.
6. Run: /etc/init.d/pisonet-agent enable
7. Run: /etc/init.d/pisonet-agent start
8. Inspect: logread -e pisonet-agent

The current build sends signed outbound heartbeats and optional signed coin-pulse events.
Captive portal and nftables traffic-control changes are intentionally disabled until board wiring and interface names are inspected over SSH.
To enable a coin acceptor later, export or provide a GPIO value file and set coin_pulse_value_path in /etc/pisonet-agent/config.json.
Most pulse coin acceptors idle high and pulse low, so coin_pulse_idle_high defaults to true.
